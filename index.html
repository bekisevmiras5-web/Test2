АРХЕОБОТ

Мы разрабатываем экосистему для безопасных раскопок: интеллект-робот, база данных находок и система визуализации.

Робот-археолог

Автономная система для точных и безопасных раскопок с использованием ИИ для идентификации артефактов.

База данных находок

Цифровой каталог всех обнаруженных артефактов с метаданными и 3D-моделями.

Система визуализации

Интерактивная платформа для изучения находок и виртуальных реконструкций.

Миссия

Наша миссия — сохранить культурное наследие при помощи технологий и обучить новое поколение исследователей.

Сохранение наследия

Использование робототехники для минимизации повреждений при археологических работах.

Образование

Разработка образовательных программ для школьников и студентов по робототехнике и археологии.

Инновации

Создание новых технологических решений для задач сохранения культурного наследия.
